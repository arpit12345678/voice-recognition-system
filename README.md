# Voice Recognition System

### Features
- Speech-to-text conversion
- Command execution
- Email sending via voice
- Opens websites 
- Search online

### Requirements
- Python 3.x
- `SpeechRecognition`, `pyaudio`, `smtplib`

### Setup
1. Fill in your email config in `config.py`
2. Run: `python main.py`

### Example Commands
- "open google"
- "send email"
- "search online"
